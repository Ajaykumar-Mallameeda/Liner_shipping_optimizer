## Image Requirements

| Property | Recommendation |
|---|---|
| Resolution | 1920×1080 for full-page, 960×540 for gallery thumbnails |
| Format | PNG (screenshots), SVG (diagrams), GIF/MP4 (demos) |
| Compression | PNG: lossless, GIF: <5MB, MP4: H.264 <10MB |
| Color Profile | sRGB |
| Aspect Ratio | 16:9 for full-page, 1:1 for badges |
